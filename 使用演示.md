- **大屏电脑**：相册宽度 800px
- **平板设备**：自动适应屏幕宽度
- **手机设备**：全屏展示，高度 450px

## 🌟 特色功能演示

### 1. 自动播放
```
点击 "▶️ 自动播放" 按钮
↓
每3秒自动翻到下一页
↓
到最后一页后自动回到第一页
↓
点击 "⏸️ 停止播放" 暂停
```

### 2. 全屏模式
```
点击 "🔍 全屏" 按钮
↓
相册占满整个屏幕
↓
黑色背景，沉浸式体验
↓
点击 "🔍 退出全屏" 返回
```

### 3. 缩略图导航
```
底部显示所有页面的缩略图
↓
当前页面缩略图有蓝色边框
↓
点击任意缩略图快速跳转
↓
鼠标悬停时缩略图放大
```

### 4. 触摸手势（手机）
```
手指在屏幕上按住
↓
向左滑动 50px 以上
↓
松开手指
↓
自动翻到下一页
```

## 📊 技术实现

### 核心技术
- **HTML5**：语义化结构
- **CSS3**：
  - Flexbox 布局
  - CSS Transform (3D 翻转)
  - CSS Transition (平滑动画)
  - 媒体查询（响应式）
  - 渐变背景
- **JavaScript ES6**：
  - Class 类封装
  - 事件监听
  - 触摸事件处理
  - 定时器控制
  - Fullscreen API

### 浏览器兼容性
- ✅ Chrome / Edge（最佳体验）
- ✅ Safari（iOS / macOS）
- ✅ Firefox
- ✅ 手机浏览器（iOS Safari, Chrome）
- ⚠️ IE 11（不支持，建议升级）

### 性能优化
- 图片使用占位符服务（按需加载）
- CSS 动画使用 GPU 加速
- 防抖处理（避免重复点击）
- 事件委托（提高性能）

## 💡 使用技巧

### 技巧1：图片优化
建议将照片压缩到 500KB 以内，加载更快：
- 使用 TinyPNG.com 压缩
- 或使用手机自带的"优化"功能

### 技巧2：图床选择
推荐的免费图床：
- ImgBB.com
- SM.MS
- 路过图床
- Imgur

### 技巧3：链接美化
如果觉得 Vercel 链接太长：
- 使用短链接服务（如 bit.ly）
- 或购买自己的域名绑定

### 技巧4：二维码装饰
生成二维码后可以：
- 添加 Logo
- 改变颜色
- 添加文字说明
- 打印成贴纸
