## 🎯 三步部署到 Vercel（5分钟完成）

### 第一步：注册账号

1. 访问 https://github.com（如果没有 GitHub 账号）
2. 注册一个免费账号
3. 访问 https://vercel.com
4. 用 GitHub 账号登录 Vercel

### 第二步：上传项目

**简单方法（无需命令行）：**

1. 在 GitHub 上创建新仓库：
   - 点击右上角 "+" → "New repository"
   - 名称：`my-flipbook-album`
   - 设为 Public
   - 创建后，上传这个文件夹里的所有文件

2. 也可以使用 GitHub Desktop（图形界面）：
   - 下载 https://desktop.github.com
   - 打开后登录 GitHub 账号
   - 点击 "Add" → "Add Existing Repository"
   - 选择这个 `flipbook-album` 文件夹
   - 点击 "Publish repository"

### 第三步：在 Vercel 部署

1. 登录 Vercel 后台
2. 点击 "Add New..." → "Project"
3. 选择你刚上传的 `my-flipbook-album` 仓库
4. 点击 "Import" → "Deploy"
5. 等待 1-2 分钟
6. 🎉 完成！你会得到一个链接

## 💻 或者使用命令行（适合会用终端的人）

如果你熟悉命令行，可以更快：

```bash
# 1. 进入项目目录
cd ~/flipbook-album

# 2. 初始化 Git（如果还没有）
git init
git add .
git commit -m "我的电子相册"

# 3. 推送到 GitHub（替换成你的仓库地址）
git remote add origin https://github.com/你的用户名/my-flipbook-album.git
git branch -M main
git push -u origin main

# 4. 安装并使用 Vercel CLI
npm install -g vercel
vercel login
vercel

# 按照提示完成部署
```

## 🎨 快速定制

### 改标题
打开 `index.html`，找到第 12 行：
```html
<h1>📸 我的电子相册</h1>
```
改成你想要的标题。

### 换照片
**使用在线图片：**
把 `index.html` 里的图片链接改成你自己的：
```html
<img src="你的图片链接" alt="照片" class="photo">
```

**使用本地图片：**
1. 把照片放到 `images` 文件夹
2. 修改链接：
```html
<img src="images/你的照片.jpg" alt="照片" class="photo">
```

### 改颜色
打开 `style.css`，第 10 行：
```css
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
```
改成你喜欢的渐变色。
