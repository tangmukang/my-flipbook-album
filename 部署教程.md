# 📚 Vercel 免费部署教程（超详细步骤）

## 🎯 部署方法选择

### 推荐：方法一 - GitHub + Vercel（最简单）
无需安装任何软件，只需浏览器操作

### 方法二 - Vercel CLI
需要一点命令行操作

---

## 🚀 方法一：GitHub + Vercel 部署（推荐）

### 第一步：注册 GitHub 账号

1. 访问 [GitHub 官网](https://github.com)
2. 点击右上角 "Sign up" 注册
3. 填写邮箱、密码，完成验证

### 第二步：上传项目到 GitHub

**方式A：网页上传（无需安装 Git）**

1. 登录 GitHub 后，点击右上角 "+" → "New repository"
2. 填写仓库名称，例如：`my-flipbook-album`
3. 选择 "Public"（公开）
4. 点击 "Create repository"
5. 在新页面点击 "uploading an existing file"
6. 将这个文件夹里的所有文件拖拽上传：
   - index.html
   - style.css
   - script.js
   - vercel.json
   - README.md
   - .gitignore
7. 点击 "Commit changes"

**方式B：使用 Git 命令（如果你会用）**

```bash
cd ~/flipbook-album
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/你的用户名/my-flipbook-album.git
git push -u origin main
```

### 第三步：部署到 Vercel

1. 访问 [Vercel 官网](https://vercel.com)
2. 点击 "Sign Up" 注册
3. **选择 "Continue with GitHub"**（用 GitHub 账号登录）
4. 授权 Vercel 访问你的 GitHub
5. 进入 Vercel 控制台后，点击 "Add New..." → "Project"
6. 找到你刚才创建的仓库 `my-flipbook-album`
7. 点击 "Import"
8. 保持默认设置，直接点击 "Deploy"
9. 等待 1-2 分钟，部署完成！

### 第四步：获取链接并分享

1. 部署成功后会看到 "Congratulations" 页面
2. 你的相册链接会是：`https://my-flipbook-album.vercel.app`
3. 点击链接测试是否正常
4. 分享给朋友或生成二维码！

## 💻 方法二：Vercel CLI 部署

### 第一步：安装 Node.js

1. 访问 [Node.js 官网](https://nodejs.org)
2. 下载并安装 LTS 版本

### 第二步：安装 Vercel CLI

打开终端（Mac）或命令提示符（Windows），运行：

```bash
npm install -g vercel
```

### 第三步：部署项目

```bash
# 进入项目目录
cd ~/flipbook-album

# 登录 Vercel（会打开浏览器）
vercel login

# 部署
vercel

# 按照提示操作：
# - Set up and deploy? Y
# - Which scope? 选择你的账号
# - Link to existing project? N
# - Project name? 按回车使用默认名称
# - In which directory? 按回车使用当前目录
# - Want to override settings? N
```

部署完成后会显示：

```
✅ Production: https://your-album.vercel.app [copied to clipboard]
```

## 🔄 如何更新相册

### 如果用方法一（GitHub + Vercel）

**更新照片或内容后：**

1. 在 GitHub 仓库页面点击要修改的文件（如 index.html）
2. 点击铅笔图标"编辑"
3. 修改内容
4. 点击 "Commit changes"
5. Vercel 会**自动重新部署**（等待1-2分钟）

或者直接上传新文件替换旧文件。

### 如果用方法二（Vercel CLI）

修改文件后，再次运行：

```bash
vercel --prod
```

## 🎨 常见自定义需求

### 1. 更换标题

编辑 `index.html` 第 12-13 行：

```html
<h1>📸 我的旅行相册</h1>  <!-- 改成你想要的标题 -->
<p class="subtitle">2024年春节旅行记录</p>  <!-- 改成你的副标题 -->
```

### 2. 增加/减少照片

在 `index.html` 中复制这段代码，修改图片链接和标题：

```html
<!-- 第7页（新增） -->
<div class="page">
    <div class="page-content">
        <img src="https://picsum.photos/600/400?random=7" alt="照片7" class="photo">
        <div class="caption">🎉 新的回忆</div>
    </div>
</div>
```

记得修改 `script.js` 第 6 行的总页数：

```javascript
this.totalPages = 7;  // 改成实际页数
```

### 3. 使用自己的照片

**如果照片在网上（推荐）：**
- 上传到图床（如：ImgBB、SM.MS）
- 复制图片链接替换 `src="..."`

**如果使用本地照片：**
1. 在项目里创建 `images` 文件夹
2. 把照片放进去（建议压缩到 500KB 以内）
3. 修改 HTML：

```html
<img src="images/my-photo.jpg" alt="我的照片" class="photo">
```

4. 上传到 GitHub 时要包含 images 文件夹

### 4. 修改颜色主题

编辑 `style.css` 第 10 行：

```css
/* 紫色渐变（默认） */
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);

/* 蓝色渐变 */
background: linear-gradient(135deg, #4facfe 0%, #00f2fe 100%);

/* 粉色渐变 */
background: linear-gradient(135deg, #fa709a 0%, #fee140 100%);

/* 绿色渐变 */
background: linear-gradient(135deg, #30cfd0 0%, #330867 100%);
```

## ❓ 常见问题

### Q1: Vercel 部署后访问很慢？
**A:** Vercel 服务器在国外，国内访问可能慢。可以考虑：
- 使用 Cloudflare Pages（类似 Vercel，国内访问稍快）
- 购买国内云服务器（阿里云/腾讯云）

### Q2: 照片加载失败？
**A:** 检查：
1. 图片链接是否正确
2. 图片是否太大（建议压缩到 500KB 以内）
3. 使用 HTTPS 链接（不要用 HTTP）

### Q3: 能不能设置密码保护？
**A:** 免费版 Vercel 不支持密码保护。可以：
1. 使用复杂的链接（不容易被猜到）
2. 使用付费服务（Vercel Pro）
3. 自己加简单的 JavaScript 密码验证

### Q4: 手机上能正常用吗？
**A:** 完全没问题！已经做了移动端适配，支持：
- 触摸滑动
- 响应式布局
- 自动缩放

### Q5: 免费额度够用吗？
**A:** 完全够用！Vercel 免费版：
- 100GB 带宽/月
- 无限项目
- 自动 HTTPS
- 对个人使用完全足够

### Q6: 能加背景音乐吗？
**A:** 可以！在 `index.html` 中加入：

```html
<audio autoplay loop>
    <source src="music.mp3" type="audio/mpeg">
</audio>
```

不过浏览器通常会阻止自动播放，需要用户先点击页面才能播放。
